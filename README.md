# Slot Machine Game (Python Console Project)

This is a simple slot machine game built using Python while I was learning programming fundamentals.

It runs in the terminal and simulates a basic slot machine where the user deposits money, places bets, and spins for results.

---

## What the project does

- Allows the user to deposit money
- Lets the user bet on 1 to 3 lines
- Generates random slot results
- Checks for matching symbols
- Calculates winnings or losses
- Keeps track of balance across rounds

---

## What I learned from this project

This project helped me understand:

- How to structure a program using functions
- How loops and conditions work together in a real program
- How to use dictionaries to store related data
- How to handle user input properly
- How to build logic step by step instead of everything at once
- How to debug errors and fix issues in working code
